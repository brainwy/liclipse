Languages supported

LiClipse supports many languages out of the box:

  • CoffeScript
  • C/C++
  • CSS
  • Dart (new on 0.9.0)
  • Django Templates
  • HTML
  • Java (new on 0.9.5)
  • JavaScript
  • Julia (new on 1.0.0)
  • Kivy Language (new on 0.9.7)
  • Mako Templates (new on 0.9.0)
  • Python
  • RST (ReStructured Text)
  • SCSS
  • StringTemplate (new on 1.1.1)
  • XML
  • YAML (new on 1.1.1)
